# How to install

	Rename file `.env.example` to `.env`. (terminal: "rename .env.example .env")

	git clone https://github.com/kafked/laravel-fsh.git

	composer install
	npm install // for unix or "npm install --no-bin-links" for windows
	npm rebuild node-sass
	gulp
